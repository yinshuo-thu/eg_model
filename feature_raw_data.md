# 数据语义与原料推断 —— 用于构造下一阶段新因子

> 目的：把对 `data.csv` 的逆向分析结论固化下来，作为后续 `tools/build_raw_factors.py` 造因子的依据。
> 所有结论均基于 `artifacts/panel_raw.parquet`（1,677,896 行 × 96 列，day 1–1259，1333 标的）实测。
> 复现脚本见文末「附录 · 复现」。**区分「已证明」与「强推断」——造因子只依赖已证明的部分即可，推断部分用于取名/直觉。**

---

## 0. TL;DR（一句话）

这是一个**个股·日频横截面选股面板**；`x_0…x_85` 是**已加工好的特征**，`prc1…prc5 / vol0 / g` 是**留给你造新因子的原料**。
- `prc1…prc5` = 同一天同一标的的 **5 个价格标记**（推断为 Open/Close/Low/High/VWAP），是**截面归一化的对数价 level**；**差分 `Δprc` 就是干净的日收益**。
- `y` = **prc2（收盘）上的「下一天」截面标准化收益**（horizon 唯一确定 = 1 天，`IC(y, Δprc2[t+1]) = 0.744`）。
- 造因子的黄金方向：**用这 5 个标记做收益分解**（隔夜/日内、close-vs-VWAP、日内区间、多周期反转）。实测这些与现有 86 个 x **近乎完全正交**，价值是"去相关弱增量"。

## 2. 列分类：哪些是 feature，哪些是 raw data

TASK1 原文："`prc1 … prc5` and `vol0` are provided as raw material for constructing ADDITIONAL features."

### 2.1 Feature（已加工的模型输入）= `x_0 … x_85`（86 个）

按分布分成 4 个家族：

| 家族 | 列 | 分布特征 | 推断 |
|---|---|---|---|
| **A 连续 alpha** | `x_0…x_42` | mean≈0、std≈0.1、有界≈±0.7、逐日截面近中性、~1.5M 唯一值 | 已截面归一化的**连续 alpha 信号**。**`x_0…x_7` 是最强单因子**（见下），符号为负 = **反转类** |
| **B 离散** | `x_43` | 类似 A，但唯一值仅 ~570K | 更离散/分档的信号 |
| **C 排序 rank** | `x_44…x_49` | std=0.577（[-1,1]均匀分布签名）、硬边界±1、ac1=**0.993** | **截面 rank 特征**（`rank/N·2−1`），极持久 → size/value/quality/beta 类**基本面排序** |
| **D 时序/滞后** | `x_50…x_85` | **每只票第一天全 NaN**（age=0 缺失率100%，之后掉到~1–2.5%） | 需要历史的**滞后/滚动派生特征**。其中 `x_60` 正值无上界（max 6560, ac 0.35）= **换手/成交额/幅度类量纲**；`x_85` 稀疏、缺失最高(7.1%) |

**各原始列对 y 的截面 IC（前几名，符号重要）：**
```
x_0  -0.0385   x_3  -0.0370   x_4  -0.0302   x_1  -0.0275   x_2  -0.0262
x_7  -0.0215   x_5  -0.0187   x_29 +0.0144   x_50 +0.0106   x_62 +0.0105
```
→ 主信号是 `x_0…x_7` 的**反转**（负 IC）。prc/vol 的**同期** IC≈0（因为它们是 level，不是收益）。

## 3. `prc1…prc5` 的语义（本文件核心）

### 3.1 已证明的事实

**① 是「同一天、同一标的」的 5 个价格标记，不是滞后。**
`corr(prc_j[t], prc1[t−L])` 对所有 j 都在 **L=0 峰值**（L1/L2 单调下降），无任一列在 L≥1 更高。

**② 每个 prc 是随机游走的价格 level；`Δprc` 是真日收益。**
- `ac1(prc)=0.967`（level）；`ac1(Δprc)=−0.03≈0`（差分=收益）
- `std(Δprc)≈2.2%`，**峰度 56–95**（股票日收益胖尾签名）
- 方差比 `VR(2/5/10)=0.98/0.95/0.93≈1`（近随机游走，略<1 = 轻微反转/买卖价跳动）
- 是**截面归一化的对数价**：逐日 cs-std≈0.0947 且几乎恒定（std-of-std 仅 0.005）

**③ 5 个标记的内部几何**（平均 |prc_i−prc_j|，×1000）：
```
       prc1  prc2  prc3  prc4  prc5
prc1    -    13.2   8.9   9.2  10.9      prc1 = 离群点（离所有列最远）
prc2  13.2    -     8.4   8.5   4.3      prc2 ≈ prc5（最近，4.3）
prc5  10.9   4.3   6.0   6.1    -        prc5 = 最居中
```
行内极值倾向：prc1/prc2 常成极值（各~30%），prc3 偏最小（mean_rank 1.72），prc4 偏最大（2.27），prc5 极少成极值（~5%）。

### 3.2 强推断：这是一套 {Open, High, Low, Close, VWAP}

| 列 | 推断 | 关键证据 |
|---|---|---|
| **prc2 = Close** | 收益基准价 | `y` = 下一日 prc2 收盘到收盘收益（IC 0.744，见 §4） |
| **prc1 = Open** | 时间上的离群标记 | 它自己的**隔日差分与 y 在 h=1 处 IC≈0，h=2 才升到 0.52** —— open-to-open 比 close-to-close 错开一天的精确签名；且离 close 最远 |
| **prc5 = VWAP** | 日内中心价 | 最居中、离 close 最近、极少成极值 |
| **prc3 = Low / prc4 = High** | 日内区间边界 | prc3 偏行内最小、prc4 偏行内最大；`range=prc4−prc3` 表现为波动/彩票折价（IC 负） |

**注意**：严格的 `High≥Close≥Low` 行内序被打破，是因为每列被**独立匿名化/归一化**过（否则 High 会恒为行内最大）。因此 OHLC 映射用于**取名/直觉**，造因子时以「§3.1 已证明的事实」为准。
**佐证**：把收益拆成 `intraday=prc2−prc1`（σ=1.96%）与 `overnight=prc1[t]−prc2[t−1]`（σ=1.16%），二者**几乎不相关(−0.03)、IC 反号**（日内 −0.0043 反转 / 隔夜 +0.0016 动量），且日内方差>隔夜方差 —— 经典的股票"隔夜/日内背离"，强烈支持 open/close 解读。

## 4. 标签 `y` 的逆向（label archaeology）

**`y[t] ≈ perday_z( prc2[t+1] − prc2[t] )` —— 下一天的截面标准化收益，horizon 唯一确定 = 1 天。**

Horizon scan，`IC(y[t], prc_j[t+h]−prc_j[t])`：
```
      prc1  prc2  prc3  prc4  prc5
h=1  -0.00  0.74  0.35  0.35  0.60   ← 峰值 0.744 在 (h=1, prc2)
h=2   0.52  0.51  0.53  0.53  0.54
h=3   0.41  0.41  0.42  0.42  0.43
```
- `h=2/3/4` 的值（0.51/0.41/0.35）≈ `0.744/√h` —— "单日信号被后续独立噪声稀释"的精确特征 → **horizon = 1 天**（不是 2–3 天）。
- `y` 与 prc 活在同一归一化空间（y 逐日 std 0.094 ≈ prc 的 cs-std 0.095），封顶 [−1,1]，峰度 7。
- IC 只有 0.74 而非 1.0：`y` 还叠加了残差化/去极值/或用了略不同的 blend 价，含约 45% 不可解释噪声。

> ⚠️ **因果红线**：上面用到 `prc[t+1]` 只是为了**辨识标签定义**，**不是**可用因子。造因子只能用 `≤ t` 的信息（`prc[t], prc[t-1], …`）。

## 5. `vol0` / `x_60` / `g`

- **`vol0` = 归一化的成交量/换手冲击**：mean +0.02、右偏、封顶 [−0.24, 1]、**ac1=0.462**（流动性有持续性）。非原始股数；与 amihud 相关 −0.386（量↑→非流动性↓，符合）。
- **`x_60`（在 x 块但行为像原料）**：正值无上界（median 0.11, 99% 0.31, max 6560, ac 0.35）= **ADV/换手/幅度**代理；`corr(log1p(x_60), vol0)=0.185`。可做**流动性分档/归一化条件**。
- **`g` = 74 个行业组**：用于中性化 / 组内相对因子；实测对反转、波动因子做 g-中性可小幅提升 IR（呼应现有 v2 的中性化 edge）。

## 6. 造新因子菜单（实测 IC，因果，`≤t`）

**两条关键事实决定策略：**
1. naive prc/vol 因子**单个都弱**（|IC| ≤ 0.007）；
2. **但与现有 x 近乎完全正交**（`corr(mom3, x_0..x_7)≈0.00–0.01`；mom3 回归掉 x_0..x_7 后 IC 从 −0.0074 → **−0.0073 几乎无损**）。
→ **它们的价值是"去相关弱增量"喂进 ensemble，不是单打独斗刷 IC。**

统一后处理：**先逐日 z-score → 可选 g-中性 → clip ±**。基准日收益用 `ret = prc2 − prc2.shift(1)`（收盘到收盘）。

| 因子族 | 定义（causal） | 实测 IC | 备注 |
|---|---|---|---|
| **多周期反转** | `−(prc2 − prc2.shift(k))`, k=1/2/3/5 | k=3 **+0.0088**；k=2 +0.0068 | 最强的一族；符号为**反转** |
| **隔夜/日内分解** | `intraday=prc2−prc1`；`overnight=prc1[t]−prc2[t−1]` | 日内 −0.0043 / 隔夜 +0.0016 | **反号、去相关**，分开喂比合成动量强 |
| **close-vs-VWAP** | `prc2 − prc5` | −0.0017 | 收盘相对 VWAP 的日内压力/反转 |
| **日内区间/波动** | `range=prc4−prc3`；`Δprc` 的 5/10/20 日滚动 std | range −0.0024；vol10 −0.0043 | 波动/彩票**折价**（负） |
| **距均线/趋势** | `prc2 − rollmean(prc2, w)`, w=10/20/60 | dma10 −0.0058 | 过度延伸反转 |
| **非流动性 Amihud** | `|Δprc2| / vol0`（clip） | **+0.0047** | 非流动性溢价（正）；与 vol0 反相关 |
| **异常量/换手** | `vol0 / rollmean(vol0,20) − 1`；`sign(Δprc)·vol0` | 弱 | OFI-lite |
| **交互** | 反转 × vol0；反转在高量名上更强 | — | 条件因子 |
| **共同/特质分解** | 对 `Δprc1..5` 做 PC1（共同）+ 残差（特质），取**特质短反转** | 待测 | 通常最干净 |

**建模建议**：把上面挑出的**正交且稳定**的几个，(a) 作为额外输入喂给 LightGBM / MLP；(b) 或独立成信号，靠 ensemble 的多样性(inverse-corr)权重接入。现实边际提升小但真实。

核心度量：
```python
def daily_ic(pred, y, day):  # 逐日截面 Pearson IC 的均值
    d = pd.DataFrame({'p':pred,'t':y,'day':day}).dropna()
    return d.groupby('day').apply(lambda s: s['p'].corr(s['t'])).mean()

def perday_z(s, day):        # 保留 Pearson IC 的逐日标准化（勿用 rank）
    g = pd.DataFrame({'s':s,'day':day}).groupby('day')['s']
    return (s - g.transform('mean')) / (g.transform('std') + 1e-9)
```

复现要点：`ret = prc2 − prc2.groupby(instrument).shift(1)` = 干净日收益；`y` 是它的**下一天**版本（截面标准化）。所有滚动/滞后按 `instrument_id` 分组、只用 `shift(≥0)`，保证因果无泄漏。
